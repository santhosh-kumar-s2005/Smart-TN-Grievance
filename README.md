# Smart TN Grievance System

This is a web application built using **Next.js** to manage and streamline grievance handling in Tamil Nadu.

---

## 🚀 Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Then open:
http://localhost:3000

You will see the application running in your browser.

---

## 🛠️ Tech Stack

* Next.js
* TypeScript
* React
* Tailwind CSS (if used)

---

## ✏️ Development

You can start editing the main page by modifying:

```
app/page.tsx
```

The page auto-updates as you save changes.
